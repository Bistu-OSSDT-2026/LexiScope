## 🧭 项目是什么

**LexiScope** 是一个轻量级的自动化文本分析工具，专为**学生、内容创作者和阅读爱好者**设计。  
你只需上传一份 `.txt` 文件或粘贴文本内容，它就能在几秒内自动完成三项核心分析：

- 📊 **高频词统计**（Top 20，过滤停用词）  
- 🏷️ **TF‑IDF 关键词提取**（Top 10，权重排序）  
- ❤️ **情感倾向判断**（得分 + 积极/中性/消极标签）

所有结果以**三栏卡片**形式清晰展示，无需手动阅读全文，即可快速把握文章要点。

---

## 💡 解决什么问题

在日常学习、阅读或内容创作中，我们常常需要：

- 快速了解一篇论文摘要或新闻稿**主要谈论什么**；
- 分析自己写的文章**风格是否偏正面**，关键词是否突出；
- 在长篇材料中**快速定位核心概念**，节省阅读时间。

传统做法是人工阅读并手动摘录，**效率低且主观性强**。  
LexiScope 通过自动化文本分析技术，**降低分析门槛**，让信息获取更加高效、客观。

## ✅ 当前版本（V1.0）能完成什么

| 功能 | 说明 |
|------|------|
| 📂 上传 `.txt` 文件 | 支持 UTF‑8 编码，大小 ≤5MB |
| 📝 粘贴纯文本 | 直接在文本框输入内容 |
| 🇨🇳 中文文本分析 | 基于 `jieba` 分词，过滤常见停用词 |
| 📊 词频统计 | 返回出现次数最高的 **20 个词** |
| 🏷️ TF‑IDF 关键词 | 手动实现算法，返回权重最高的 **10 个词**（保留 3 位小数） |
| ❤️ 情感分析 | 内置 60+ 情感词库，输出 **得分 + 标签（积极/中性/消极）** |
| 🖥️ 卡片式结果展示 | 三栏横向布局，响应式设计，适配桌面与移动端 |
| ⚠️ 友好错误提示 | 空文件、非 txt 格式、编码错误等均有明确反馈 |

### ❌ 本次不包含（后续迭代）
- 支持 `.docx` / `.pdf` 等格式
- 批量文件分析
- 词云可视化
- 文章自动摘要
- 接入大模型 API
- 结果导出为 PDF/Word

## 🛠️ 技术架构

- **后端**：Flask（Web 服务）
- **分词**：jieba（仅用于词频模块）
- **算法实现**：纯 Python 内置库（re, math, collections）
- **前端**：原生 HTML + CSS + JavaScript（单页应用，无外部依赖）
- **部署**：单文件 `app.py`，开箱即用

### 模块划分

| 模块 | 负责同学 | 核心函数 | 说明 |
|------|----------|----------|------|
| 前端界面 | E | `index_html()` | 页面结构、交互 JS、数据渲染 |
| Web 服务 | A | `create_app()` / `analyze()` | 路由、文件接收、调用各模块、返回 JSON |
| 词频统计 | B | `get_word_freq(text)` | 中文分词、停用词过滤、频次排序 |
| 关键词提取 | C | `extract_keywords(text)` | 手写 TF‑IDF 算法（句子级） |
| 情感分析 | D | `get_sentiment(text)` | 硬编码情感词库，计算得分与标签 |

## 🚀 如何运行与使用

### 1. 环境要求
- Python 3.8 或更高版本
- 建议使用虚拟环境（如 `venv`）

### 2. 安装依赖

```bash
pip install flask jieba
```

> 如果网络较慢，可使用国内镜像：  
> `pip install flask jieba -i https://pypi.tuna.tsinghua.edu.cn/simple`

### 3. 获取代码

将本项目所有代码合并至一个 `app.py` 文件（或直接克隆本仓库）：

```bash
git clone https://github.com/your-username/LexiScope.git
cd LexiScope
```

### 4. 启动服务

```bash
python app.py
```

终端会显示：
```
 * Running on http://127.0.0.1:5000
```

### 5. 使用

打开浏览器访问 `http://127.0.0.1:5000`，你会看到：

- **上传区**：点击选择 `.txt` 文件，或直接粘贴文本到文本框；
- **开始分析**：点击按钮，等待片刻；
- **结果展示**：三栏卡片自动刷新，显示词频、关键词和情感信息。

> 💡 **提示**：如果上传空文件或非 txt 格式，页面会显示红色错误提示。
